📋 Core Use Cases

Use Case 1: Legal Document Query

Actor: Legal Professional/Citizen

Goal: Get specific legal information

Flow:

  • User submits legal question via API
  • System classifies query as legal/non-legal
  • System retrieves relevant documents from vector database
  • AI generates comprehensive legal response
  • Response delivered with optional audio/TTS

Use Case 2: PDF Document Processing

Actor: Document Administrator

Goal: Make legal documents searchable

Flow:

  • Upload PDF legal document
  • System extracts and chunks text content
  • Generate embeddings and store in Pinecone
  • Document becomes searchable for future queries

🔧 Advanced Use Cases

Use Case 3: Web Crawling for Current Laws

Actor: System (Automated)

Goal: Access most recent legal information

Flow:

  • System detects need for current information
  • Identifies relevant legal area from query
  • Crawls government websites (chinhphu.vn)
  • Extracts and processes PDF documents
  • Stores in temporary vector store
  • Incorporates findings into response

Use Case 4: Multi-language Legal Assistance

Actor: International User

Goal: Access Vietnamese legal information in English

Flow:

  • User asks question in English
  • System translates query to Vietnamese for document search
  • Retrieves Vietnamese legal documents
  • Generates response in original user language

🎵 Audio & TTS Use Cases

Use Case 5: Audio Legal Consultation

Actor: Audio-preferred User

Goal: Receive legal guidance through audio

Flow:

  • User requests audio response
  • System processes legal query normally
  • Breaks response into manageable chunks
  • Generates TTS audio with gTTS
  • Creates lip-sync data for visual representation
  • Streams audio chunks with synchronization
